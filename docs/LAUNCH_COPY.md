## Hacker News draft

**Title:** Show HN: Governance checks for scoped AI coding-agent changes

I am the author of Coding Agent Governance. It turns a bounded AI coding task into a TaskContract, checks the actual change scope, records task-relevant test evidence, and blocks Closure when verification becomes stale. The demo is synthetic-only and offline: one denied diff, one stale verification, and one successful Closure. `CLOSED != PRODUCTION_READY`; the result is an evidence-backed task closure, not a production-readiness certification. https://github.com/vanlew1/ai-agent-project-governance
